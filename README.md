
# review-dapp
vue仿DAppReview官网的一个练手项目
# 在线查看本项目效果 [查看]()
## 食用方法

### 第一步 ：添加作料

npm install 

### 第二步 ：爆炒

npm run dev 

### 第三步 ：出锅

在浏览器打开 localhost:8080 即可食用
